# Competitive Wordle
Real-time multiplayer Wordle clone built with Spring Boot, React, WebSockets, and JWT authentication.

## Tech Stack

Backend:
- Spring Boot
- Spring Security + JWT
- WebSocket (STOMP)
- JPA / MySQL

Frontend:
- React
- TypeScript (if using)
- STOMP client

Infrastructure:
- Docker (planned)

## Running Backend

cd backend
./mvnw spring-boot:run
